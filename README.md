# Todo

A todo app for daily tasks.

## Features list

- [ ] Task with title and description.
- [x] Add task.
- [ ] Delete task.
- [ ] Edit task.
- [x] Timestamp when created.
- [x] Timestamp when to be done.
- [ ] Save locally.
- [ ] History.
- [ ] Mark as done.
- [ ] Warn when not done when time reached.
- [ ] Download task as pdf or others.

- HTML
- CSS 
- JS
